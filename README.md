# Getting Started

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

To install dependencies run

```bash
npm install
```

Then to run the application on localhost:3000 run

```bash
npm start
```

To run tests run

```bash
npm test
```
